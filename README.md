# Heap Sort Visualizer

**Interactive visualization of the Heap Sort algorithm using Max-Heap data structure.**  
Watch each swap and heapify operation step by step, with colored bars, speed control, and detailed logs.

---

## Features

- **Real-time visualization** – See bars grow and swap with color feedback
- **Step-by-step logging** – Every swap is recorded with array state
- **Custom array input** – Enter any comma‑separated integers (≤50 values)
- **Random array generator** – Instantly generate random test data
- **Adjustable speed** – Slow down for learning or speed up for overview
- **Complexity info** – Built-in O(n log n) explanation
- **Modern dark theme** – Easy on the eyes during long sessions
